# Asteroids Game

Un clone du jeu classique Asteroids réalisé avec Pygame.

## Prérequis
- Python 3.6+
- Pygame 2.0+

## Installation
```bash
git clone https://github.com/votreuser/AsteroidsGame.git
cd AsteroidsGame
pip install -r requirements.txt
```

## Exécution
```bash
python asteroids.py
```

## Contrôles
- Flèches : Déplacement
- Espace : Tirer
- R : Redémarrer
